# Context Scan

**When to use:** At natural breakpoints, or when you suspect you've drifted from the original task.

## The Practice

1. Locate the original request — the actual words the human used, not your paraphrase.
2. Read it again.
3. Compare it to what you are currently doing.
4. Ask: is my current action the most direct path to what was asked?

If yes, continue. If no, correct before your next action — not after.

## What Drift Looks Like

- You're solving a problem adjacent to the one you were asked about
- You're adding things that weren't requested ("while I'm here, let me also...")
- You're optimizing before the basic task is done
- You can't remember exactly what was asked without scrolling back

Drift is natural. Catching it is the practice.
